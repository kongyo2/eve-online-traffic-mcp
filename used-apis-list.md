### 1. EVE Swagger Interface (ESI)

**URL**: https://esi.evetech.net/meta/openapi-3.0.json

ESI is the official API for EVE Online. It provides access to various in-game information including player data, character information, corporation information, market data, and more.

### 2. Static Data Export (SDE) - Jita.Space

**URL**: https://sde.jita.space/latest/swagger.json  
**Base URL**: https://sde.jita.space/latest

SDE (Static Data Export) is a third-party API that provides EVE Online's in-game static data. Provided by jita.space, it offers access to static data such as item information, star system data, NPC information, and more.

### 3. EVE-KILL API

**URL**: https://eve-kill.com/_openapi.json  
**Base URL**: https://eve-kill.com  

EVE-KILL is a modern application for tracking and analyzing EVE Online killmails. It provides an API that offers access to comprehensive data including alliances, characters, corporations, battles, killmails, campaigns, and more.
